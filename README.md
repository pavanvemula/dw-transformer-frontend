# dw-transformer-advanced (Frontend - React + Vite)

Advanced DataWeave 2.0 Code Generator UI (Version 2)

## Features
- Supports JSON, XML, CSV input/output
- File upload for input/output
- Auto-detect formats and convert CSV/XML to JSON internally
- Rule-based mapping inference (map, concat example)
- Download generated .dwl file
- Dark mode, copy to clipboard

## Quickstart

1. Install dependencies
```
npm install
```

2. Copy env example
```
cp .env.example .env
```

3. Run dev server
```
npm run dev
```

Make sure your backend is running at the URL in `.env` (defaults to http://localhost:3000)
